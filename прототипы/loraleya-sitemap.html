Технические заметки

  • Платформа: WordPress + WooCommerce, кастомная тема
  • Покупка без отдельных карточек товара — добавление в корзину со страниц сценария/цвета
  • Оплата: ЮKassa (карты, SBP) или Cloudpayments
  • Доставка: СДЭК API + Boxberry API + Почта России
  • Индивидуальный заказ: форма → Telegram-бот + email
  • Мобильная версия — приоритет (70%+ трафика)
  • SEO: страницы цветов = посадочные для поисковых запросов
  • Контент: 17 цветовых страниц × (видео + фото + описание)
